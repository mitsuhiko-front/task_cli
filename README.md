#  Task Management API

FastAPIを用いたタスク管理APIです。  
ユーザー認証（JWT）を行い、ユーザーごとにタスクを管理できます。

---

##  デモ

Renderでデプロイ済み  


## 🛠 技術スタック

- Python
- FastAPI
- PostgreSQL
- psycopg2
- JWT認証
- pwdlib (パスワードハッシュ化)
  

---

##  機能一覧

### 認証
- ユーザー登録
- ログイン（JWT発行）

### タスク管理
- タスク作成
- タスク一覧取得
- タスク更新
- タスク削除（論理削除）

### その他
- ユーザーとタスクのJOIN取得
